非官方淘宝API
====

# Features
* 获取订单
	* 获取未发货订单
	* 获取退款订单
* 订单解析
	* 获取商品
	* 获取收货地址
	* 解析收货地址中的手机号
* 订单发货
	* 虚拟商品发货(无物流)
* 退款管理
	* 拒绝退款
	* 添加退款说明与附加图片

# Dependencies
* python3
* beautifulsoup4
* selenium
* firefox
* geckodriver
